FLVTool2 is a manipulation tool for Macromedia Flash Video files (FLV). It calculates various meta data and inserts an onMetaData tag in the video. It also cuts FLV files and adds cue Points (onCuePoint). A debug command shows the inside of an FLV file and the print command gives meta data information in XML or YAML format.

Shared installation

By default, DreamHost users have access to a shared installation of FLVTool2.

In web applications, the absolute path to the shared installation of FLVTool2 is:

/usr/bin/flvtool2
