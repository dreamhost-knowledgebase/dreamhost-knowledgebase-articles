Overview

BoldGrid (aka, 'WP Website Builder') is a simple-to-use WordPress site builder. It uses a drag & drop editor, which allows you to quickly build and customize a professional website without the need for coding.

BoldGrid comes pre-configured with a list of mobile-friendly themes for you to choose from depending on your site's needs. Additional tools such as backups are available as well to ensure your data is safe.

Enabling BoldGrid on your website

There are three options to enable BoldGrid on your WordPress website. 

Option 1 — Sign up for BoldGrid when opening a DreamHost account

View the following article to open a new DreamHost account.

  • How to sign up for a DreamHost account

Make sure to check the box titled WP Website Builder. This installs BoldGrid within your WordPress installation.

Option 2 — Enable BoldGrid on an existing WordPress installation

If you didn't originally install BoldGrid when setting up your DreamHost WordPress plan, you can still enable it within the panel.

Disable other editors

BoldGrid should not be used if another website builder, such as Elementor, is already in use. If your site was built using one WordPress website builder and you want to change to another, you should first use a staging site instead of installing both builders on the live site.

  1. Navigate to the Install WordPress page.
  2. Click the Manage my Apps button.
  3. Click your domain name to expand its settings.
  4. Under the Actions column, click the Install WP Website Builder button.
    A confirmation pop-up box opens asking to confirm the installation.
  5. Click the Install button to confirm. WP Website Builder will become available in your WordPress installation in a few minutes.

Option 3 — New WordPress installations

This option installs BoldGrid when installing WordPress in the panel. View the DreamHost WordPress installer article for full installation instructions.

After you click Install WordPress, the next page allows you to select a site purpose for a guided WordPress install.

To install BoldGrid:

  1. Choose the Other option.
  2. To the right, click the Next button.
  3. When the plugins page appears, make sure WP Website Builder is checked.
  4. Click Install to proceed with installing a new WordPress site with BoldGrid.

Managing your BoldGrid website

Premium key

After enabling BoldGrid using options 2 or 3 above, a BoldGrid Premium Connect Key is installed.

To confirm if a Premium Connect Key was installed, log into your WordPress site and navigate to the Dashboard tab in the left pane. On the right side under BoldGrid Notifications, look for Boldgrid Connect Key. If a key is not installed, contact DreamHost support.
