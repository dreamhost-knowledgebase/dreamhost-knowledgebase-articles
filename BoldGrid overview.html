Overview

BoldGrid is a simple-to-use WordPress site builder. It uses a drag & drop editor, which allows you to quickly build and customize a professional website without the need for coding.

BoldGrid comes pre-configured with a list of mobile-friendly themes for you to choose from depending on your sites needs. Additional tools such as backups are available as well to ensure your data is safe.

How can I enable BoldGrid on a WordPress installation?

If you didn't originally sign up for BoldGrid on your DreamHost WordPress plan, you can still enable it two different ways through your web panel.

After enabling BoldGrid using one of the methods below, a BoldGrid Premium Connect Key is installed.

To confirm if a Premium Connect Key was installed, from your WordPress dashboard click the Dashboard tab in the left pane. On the right side under BoldGrid Notifications, look for Boldgrid Connect Key. If a key is not installed, contact DreamHost support.

Existing WordPress installations

BoldGrid should not be used if another website builder, such as Elementor, is already in use. If your site was built using one WordPress website builder and you want to change to another, you should first use a staging site instead of installing both builders on the live site.

If you have an existing WordPress installation:

  1. Navigate to the One-Click Installs page.
    The Manage Installed Applications button appears near the top of the One-Clicks Installs page in your panel:
  2. Click the Manage Installed Applications button.
  3. Expand the domain by clicking the arrow icon on the left.
    The One-Click Install expands for that domain, which shows you additional options:
  4. Click the Install WP Website Builder button on the right.
    A confirmation pop-up box opens asking to confirm the installation:
  5. Click the Install button to confirm. The WP Website Builder will become available in your WordPress installation in a few minutes.

New WordPress installations

If you don't have a WordPress installation yet:

  1. Navigate to the One-Click Installs page.
  2. Select WordPress from the list that appears.
    The following setup box appears when you select WordPress:
  3. Make your selections in the drop-down boxes and check the 'Install WP Website Builder' check box.
  4. Click the Install it for me now! button when finished.
