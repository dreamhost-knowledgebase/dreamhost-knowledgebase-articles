Before you get started

Before you begin using Remixer's image editor, please note the following:
  • You must use an image 'Web Section' in order to place an image on your page. Images can only be placed within the pre-defined area of that section. It's currently not possible to upload images to any location outside of an image 'Web Section'.
  • It's currently not possible to resize image file sizes in Remixer.
  • To improve your site speed, make sure your image file sizes are optimal. A general size would be 300k for logos and 100k for all other images. The more images, the longer your page will take to load.
  • For image editing, you can zoom, flip, or rotate images, and apply color filters.

Uploading images into Remixer

Remixer allows you to upload images from several sources such as social media accounts, Pixabay, or an existing website.

Not all image formats function with Remixer. For example, if you try to upload a .svg or .tiff file, Remixer will display an error message.

It's recommend you use .png or .jpg image formats.

You can also add a .gif images, but they will display as static images even if the gif itself is a moving image.

Once uploaded, you can add them to your website.

To upload:

  1. Open Remixer.
  2. Click the 'Media' button on the top right.
  3. Click the Images link to view your options to upload.
    • Upload from computer
    • Import social media
    • Search Pixabay
    • Import a website
  4. Click on any option and upload your images.

Adding an image to your page

  1. First, you must add a 'Web Section' that contains images.
  2. After the 'Web Section' has been added, click the stock image you'd like to change.
  3. The left pane opens an image editor. Here you can click an image you've already uploaded.
  4. Once clicked, the image is immediately added to your page.

Deleting an image

  1. Click the Media button on the top right.
  2. Click the Images link to view all images.
  3. Click the image.
  4. On the bottom right, click the red 'Delete' button.

Editing an existing image on your page

After you've added a ''Web Section'', uploaded your images to the 'Media' page, and added your image to your Remixer page, you can continue to edit the image further.

Remixer allows you to:

  • flip
  • rotate
  • center
  • adjust opacity, and
  • adjust image scale
  1. Click the image to open the left editing panel.
  2. The current image displays in the left panel where you can make adjustments to it such as rotate, flip, and re-center. Below these options are sliders for adjusting Image Opacity and Image Zoom Scale.
  3. To replace the image with one from your collection, click the Library tab. Then click an image in the list of images below. If your image selection is not in the Library yet, click Add Media button to upload your image directly. Alternatively, you can upload your images under “Media” in the top right navigation bar as described above.
  4. Changes in the Image Editor are automatically applied. If you wish to change a previous image edit, click the Undo/Redo button in the top left corner of the Editor panel.

Adding images from Remixer's royalty-free catalog (Pixabay)

If you do not have any stock images, you can find some from Remixer's royalty-free image catalog. Click the Stock tab in the Image Editor to preview a list of stock photos. Then click the View More Images button to expand your search by specific keywords (e.g., "cat"):

Once you have selected stock photos from Pixabay search, it is added to your Library (or, Media) catalog. To return to the Image Editor, click the Close button in the bottom left corner of the Pixabay Search page.

You can now see the imported image by clicking the Library tab within the current image editor.

Applying a hyperlink to an image

You can also apply a hyperlink to your image by clicking the Link tab. Select the type of link you want from the drop-down menu:

  • URL — Type in the URL you want the image to redirect to.
  • Anchor — Link to any 'Web Section' on the current page
  • Email — Set an email address as a mailto:// link
  • Page — Link to another Remixer page

Please note that Image hyperlinking does not work for Hero or Parallax fullscreen 'Web Sections'. You can tell which 'Web Section' you're in by looking at the top right of the section. You'll see the name of it.

Exiting the Image Editor

After you finish editing your images, click the Close button in the bottom right corner of the Image Editor panel.
