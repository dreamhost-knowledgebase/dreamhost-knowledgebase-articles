The Bulk-Edit feature in the panel allows you to create multiple forward addresses at once. This saves you time as you do not need to create each forward address individually.

  1. Navigate to the (Panel > 'Mail' > 'Manage Email') page.
  2. At the top is a list of your domains. Click the domain you wish to create forwards on.
  3. Click the Show Selected Domains button to view all email addresses already created on the domain.
  4. Click the Bulk-Edit Forwarding-Only Addresses button.
  5. In the text box, enter a forward address followed by the address it forwards to.
  6. Repeat for every forward you'd like to create.
  7. When finished, click the Save Changes button.
  8. Back on the 'Manage Email' page you can see your new forward addresses.
