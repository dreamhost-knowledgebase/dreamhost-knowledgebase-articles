Simple Object Access Protocol (SOAP) is a protocol specification that uses XML to communicate information between different types of systems. View SOAP on Wikipedia for further information.

Is SOAP available at DreamHost?

Yes, SOAP is available by default on all DreamHost web servers.

You do not need to set up any special configuration to run SOAP with your PHP-hosted domain(s).
