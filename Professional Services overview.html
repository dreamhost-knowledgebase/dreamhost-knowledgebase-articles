Overview

DreamHost provides several services for making changes to configurations that would typically be handled by a developer or website maintainer. If you find yourself in a situation that requires changes to your site, and your developer isn't available to help, DreamHost's skilled support team may be able to pitch in for a small fee.

To request one of the services listed below, please submit a ticket on the Contact Support page indicating the specifics of your request, including the domain and professional service you're interested in.

WordPress

Changing WordPress site URLs

Price: $19.99

If you have changed your domain name, or are switching to an https-enabled site, the DreamHost support team can change your WordPress site URL for you.

To submit a WordPress URL change request, contact support. Please include the domain/URL you are changing from, and the new domain/URL you would like to use. The DreamHost support team will reply back confirming the $19.99 charge and work being done.

Update WordPress

Price: $19.99

If you are using an older WordPress version and are worried about running into problems when updating to a newer version, DreamHost support can update it for you.

DreamHost support is able to update all themes and plugins that are part of the official WordPress.org theme and plugin repositories.

To submit a WordPress upgrade request, contact support. Please include the site you'd like to upgrade. The DreamHost support team will review your site to ensure it meets the eligibility requirements, and reply back confirming the $19.99 charge and work being done.

WordPress database optimization

Price: $19.99

Over time, a WordPress database can accumulate unnecessary overhead that can slow down your site or otherwise impact site performance. If this occurs, DreamHost support can optimize your WordPress database for you.

To submit a request to have your WordPress site's database optimized, contact support. Please include the site and, if possible, the database you'd like optimized. The DreamHost support team will review your site and database and confirm the $19.99 charge for the service.

Importing WP XML / demo data

Price: $19.99

If you have an XML backup of your site data (from wordpress.com or similar), DreamHost can import the XML data for you.

If you need further customization or actual setup of the domain itself, or even MySQL import, then you should consider DreamCare to help you with that.

PHP and MySQL

Custom php.ini values

Price: $29.99

Different PHP-based applications may have specific php.ini values that need to be customized in your site's phprc file. If you've never made these changes, this can get a little confusing and become difficult to do on your own. If you need specific PHP values, contact DreamHost support and request this service.

Contact support to submit a request to modify your php.ini values. Please include the site and exact changes you require, or a link to the system requirements for the application that needs to be changed. The DreamHost support team will review your request, the work needed, and then confirm the $29.99 charge for the service.

MySQL database import

Price: $19.99

If you are importing data from a previous site or hosting company, you may need a MySQL database imported to your DreamHost account.

To submit a request to import a MySQL database, contact support. Please include the location of your .sql export, and any information you feel the team may need while importing the database. The DreamHost support team will review your request and confirm the $19.99 charge for the service.

Installing third-party webapp on a domain

Price: $29.99

If you want to use a third-party web application, and are not comfortable using the command-line (SSH), the DreamHost Professional Services can help you install it.

This is only available for straightforward PHP-based installations. You should instead consider DreamCare for for more complicated applications.

DNS

Adding Custom DNS Records

Price: $19.99

If you need help adding a DNS record in your DreamHost account to direct your domain to another host or service, DreamHost support can do that for you.

To submit a request to add custom DNS records to your DreamHost account to utilize other services, contact support. Please include the domain and the DNS records you wish to change. The DreamHost support team will review your request and confirm the $19.99 charge.

SSL

Importing external SSL for customer

Price: $19.99

If you have an external SSL certificate and can't figure out how to properly paste it into the DreamHost Web Panel, the DreamHost Professional Services team can help you with that.

DreamCare

DreamCare is a specialized professional service that provides custom web development, database administration, and engineering solutions. DreamCare offers a resolution to errors that are typically beyond the scope of DreamHost's standard Technical Support.

If this is a service you think you may be interested in, please contact support with more information about the custom work that you wish to accomplish.
