Overview

The following describes how to add an email account on an Android device using the Outlook app.

View the following articles for a list of acceptable server settings:

  • Email Client Configuration
  • Email client protocols and port numbers

For a list of alternate mail apps, view the See also section at the bottom of this page. However, the settings are generally the same for any app you choose.

Configuring your email

  1. Make sure you have installed the Outlook app from the Google Play Store. Once installed, open the mail app.
  2. Press the Add Account button.
  3. Enter the email address you wish to connect to.
  4. Press the Continue button.
  5. Choose which type of account you'd like to use. IMAP is recommended.
  6. Click the Advanced Settings toggle at the bottom of the page.
  7. Enter the following under the IMAP / POP3 Incoming Mail Server settings:
    • IMAP / POP3 Host Name – View the Email server name article to ensure you're using your correct mail server name. This should be either imap.dreamhost.com or pop.dreamhost.com.
    • Port – This should be either 993 when using IMAP or 995 when using POP3.
    • Security type – Select SSL/TLS from the dropdown.
    • IMAP / POP3 Username – Your full email address
    • IMAP / POP3 Password – The password of your email address
  8. Enter the following SMTP Outgoing Mail Server settings:
    • SMTP Host Name – This should always be smtp.dreamhost.com.
    • Port – In most cases, you should use 465.
    • Security type – Select SSL/TLS from the dropdown.
    • SMTP Username – Your full email address
    • SMTP Password – Your password
  9. Click the checkmark button to complete the configuration.

Editing your email configuration

Once your email address has been added, you cannot edit the configuration settings. To edit your email configuration, you must delete the email address from Outlook and add it again. The following instructions describe how to delete your email address from Outlook.

  1. Open the Outlook app.
  2. Click the envelope icon.
  3. Click the gear icon.
  4. Select your email account under Mail accounts.
  5. Select the DELETE ACCOUNT option.
  6. Tap the DELETE button to remove the email address.
  7. Add the email address back to the app using the new configuration details described above.
