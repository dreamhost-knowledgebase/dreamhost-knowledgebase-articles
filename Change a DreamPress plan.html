Overview

This article describes the various ways you can upgrade or downgrade from a DreamPress hosting plan.

Downgrading to Shared Starter

If you would like to downgrade from a DreamPress plan to a Shared Starter hosting plan, make sure you fall within the limitations described in the following article:

  • Adding a Shared Starter hosting plan

If you would like to proceed, contact support to request this.

Downgrading to another hosting plan

First, make sure you've added the hosting plan you'd like to move to.

  • Adding a Shared Hosting plan
  • Adding a private server (VPS)
  • Add a Dedicated server

You can then use the following article to move off DreamPress to this new plan.

  • Downgrading DreamPress to another hosting plan

Changing to a different DreamPress plan

View the following article for instructions on how to change your DreamPress hosting plan.

  • Changing a DreamPress plan
