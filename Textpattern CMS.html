As part of an effort to better maintain and support One Click Install applications, DreamHost is no longer able to provide new installations or upgrades for Textpattern CMS through the DreamHost panel. The following provides information on how to maintain and where to find further information on upgrading to the latest version.

How do I continue to maintain Textpattern CMS?

In order to ensure your Textpattern CMS installation remains secure and compatible with the latest versions of PHP, DreamHost strongly recommends bookmarking and regularly checking the official Textpattern CMS site, here.

How can I find the latest version of Textpattern CMS?

The last version offered by DreamHost was 4.5.7, and the latest version available from the developer is 4.6.2 (which you can download here). It is advisable to upgrade as soon as possible using the instructions from Textpattern CMS, here.
