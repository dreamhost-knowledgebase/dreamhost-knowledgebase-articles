Manage Domains

The (Panel > 'Domains' > 'Manage Domains’) page allows to you manage the following features for your domains.

The ‘Manage Domains’ page allows you to manage your domain’s hosting. Here you can add hosting and adjust the domain’s settings, and you can also restore a domains web files from here as well.

View the following article for further details:

  • Domains panel overview

Registrations

The (Panel > 'Domains' > 'Registrations’) page is where you can register a new domain and manage current domain registrations you have with DreamHost. All WHOIS modifications (including nameservers) can be made from here, as well as registration renewal settings and domain locking.

View the following article for further details:

  • Domain Registration

Reg. Transfer

The (Panel > 'Domains' > 'Reg. Transfer’) page is where you can transfer a domain registration to or from DreamHost, as well as obtaining the domain’s authorization code.

View the following articles for further details:

  • Transfer your domain registration away from DreamHost
  • Transfer your domain registration to DreamHost

Secure Hosting

On the (Panel > 'Domains' > 'Secure Hosting’) page, you can add an SSL certificate to your domain, which allows the domain to be accessed using the HTTPS protocol.

For further information view the following article:

  • Secure Hosting overview

Remap Sub-Dir

The (Panel > 'Domains' > 'Remap_Sub-Dir’) page allows you to remap a directory, which is useful for giving other users access in order to alter specific subdirectories without needing to give them full access to the site.

For more information, view the following article:

  • Remap Sub-Directory

Anonymous FTP

The (Panel > 'Domains’ Anonymous FTP’) page is where you can create an FTP service which allows anonymous users the ability to upload to your website.

For further information view the following article:

  • Anonymous FTP

Mongrel and Proxy

These features are currently available for DreamHost VPS and Dedicated servers only. From the (Panel > 'Domains' > 'Mongrel and Proxy’) page, you can create a Mongrel application server to host your Ruby web applications, including Ruby on Rails. When you create a Mongrel server, an appropriate proxy server is automatically created and configured for you, and you can also create your own proxy as well.

For more information and setup instructions, please visit the following articles:

  • Mongrel
  • Proxy Server

Site Statistics

You can enable site statistics on the (Panel > 'Domains' > 'Site Statistics’) page. The site statistics application runs through a program called Analog and provides various types of reports, including statistical reports for your domain.

View the following links for further details:

  • Statistics Overview
  • Analog
  • Accessing Stats
