Connecting to a Dedicated server

If your Dedicated server's name is ds12345, you can connect to it before adding any domains on it by using the following hostname:

ds12345.dreamservers.com

Once your domain is added, you can use that for the hostname. But, you can continue to use the servername if you wish. View the following articles for information on how to log into your server:

  • FTP overview and credentials
  • SSH overview
