Overview

This article explains how to change the password assigned to your Dedicated server's admin user.

Changing your password

  1. Navigate to the Admin Users page.
  2. Click the Login info button to the right of the admin user.
    The user information box expands.
  3. Click the Change password button.
  4. Choose one of the following options:
    • Generate strong password — click this link to create a system-generated password, or
    • Password field box — manually enter a new password.
  5. Click the Reset Password button to save.
    When the password successfully updates, a success message displays.

Please allow up to five minutes before logging in.
