Actions you can perform in either grid or list view

Depending on your website's settings, you can make various adjustments in either Grid or List view. The following describes all of the options that are available in each view.

Grid

Description of Grid items

  • Link to your website — Hover over your website's URL shown as exampledreamhost.com and example.com in the above examples. Anicon appears next to it. Click it to open the selected website. The website URL also turns blue on hover.
  • or (SSL certificate) — Shows either a () locked padlock or an () un-locked padlock. A locked (green) padlock indicates that the website has an SSL certificate installed and is secure, and an un-locked (red) padlock indicates that no certificate is installed and is not secure. When you click the un-locked padlock, a dialogue box opens with a link to the SSL/TLS Certificates page.
  • Website hosting plan (shown as DreamPress Advanced and Shared Unlimited in the above examples) — The type of hosting plan appears beneath the website's URL. For example: Shared Unlimited, VPS, DreamPress, or Remixer (to name a few). It can also show the domain settings for the website, such as Mirror, DNS Only, or Redirect.
  • (WordPress) — If it's a WordPress site, thelogo appears in the upper right of the grid box. Click the WordPress button to open the wp-admin page for your WordPress site.
  • (Email) — Theicon shows whether Email is set up on the website or not, and if so, how many email addresses are added to it.
  •  (DreamShield) — Thelogo appears if DreamShield Malware Remover is installed.
  • Manage button — Hover over your website, and the Manage button appears. Opens either the Manage Websites page for your website or the Managed WordPress page (if it's a DreamPress site).
  • (Files) — Opens the file manager for the website.
  • Edit Website (not shown in the above example) — If it's a Remixer site, the Edit Website button appears, which opens the Remixer panel.
  • (not shown in the above example) — If there's no hosting plan on your site, the Add Hosting button appears, which opens the Add Hosting page, where you can add hosting to a domain. See the Adding a website and hosting article for more information.
  • (more information) — Click the three-dot () icon to open a menu that provides links to various areas of the DreamHost web panel, including Domain Registration, DNS Settings, Hosting Plan, Manage Email, and more.

List

Description of List-column items

  • Domain Name — The URL for your website. Hover over your website's URL (anicon appears next to it) and click it to open the selected website. The website URL also turns blue on hover.
  • Registration — The expiration date for your website's domain.
  • Hosting Plan — The type of hosting plan for your website. For example: Shared Unlimited, VPS, DreamPress, or Remixer (to name a few). It can also show the domain settings for the website, such as Mirror, DNS Only, or Redirect.
  • PHP — The PHP version currently set on your site.
  • SSL — Shows either a () locked padlock or an () UN-locked padlock. Locked (green) indicates that the website has an SSL certificate installed and is secure, and un-locked (red) indicates that no certificate is installed and is not secure. When you click the unlocked padlock, a dialogue box opens with a link to the SSL/TLS Certificates page.
  • DreamShield — Indicates whether DreamShield Malware Remover is installed () or not installed (). If DreamShield is not available on a domain (e.g., it's DNS only, Redirect, or otherwise does not have hosting on it), then a '' mark appears in this column.
  • Email — Indicates the number of email addresses set up on the domain. A '0' indicates that no email addresses are set up yet.
  • Links — There are several types of icons that appear in the Links column:
    • — Opens the Manage Domains page, where you can add a hosting plan.
    • — Opens the DNS page, where you can make adjustments to your domain's DNS records.
    • — Opens the file manager for your website , where you can connect to your webserver to view, upload, download files, and more.
    • — Opens Remixer.
    • — Opens the wp-admin page for your WordPress site.
  • Columns — You can adjust which column items to display by clicking the  icon on the far right.
  • Manage — Opens either the Manage Websites page for your website or the Managed WordPress page (if it's a DreamPress site).
  • More information () — Click the three-dot () icon on the far right, which opens a menu that provides more links to various areas of the DreamHost web panel, including Domain Registration, DNS Settings, Hosting Plan, Manage Email, and more (options available are based on the current hosting plan).
