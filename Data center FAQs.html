What is a data center?

A data center is a facility that houses different types of servers such as website servers, database servers, mail servers, and cloud servers. If you have purchased hosting services from DreamHost, your website data is stored and protected within a data center.

Where are DreamHost's data centers located?

DreamHost currently has data centers in Ashburn Virginia and Hillsboro Oregon.

Where is my data center?

You can view which data center your services are located in within your panel. Navigate to the Data Centers page. This page lists all of your services and the corresponding data center they are located in.

Does it matter where my data center is located?

The only time this is really an issue is if your website uses a database. This is because your web server and database server should always be in the same data center.

Occasionally, the web server and database server could be placed in different data centers. This could lead to slowness on your site since it takes a little longer for the information to be retrieved from the other data center.

If you notice this, contact support so they may migrate your servers to the same location.

Can I change my data center?

Yes. Contact support if you'd prefer your information be located in a different data center.
