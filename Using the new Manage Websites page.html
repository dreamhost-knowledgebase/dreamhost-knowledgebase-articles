Overview

Related article: Websites panel page

When you click the Manage button on the Websites page, the Manage Websites page opens (except for DreamPress and Remixer plans, which instead link to their own manage page). 

The following describes all of the items and action you can take on the Manage Websites page.

Upper display

The upper display provide top-line information about your website, navigational tools, and two tabs (Website and Hosting) that provide more detailed information.

Each of these are described below.

Return to the Websites page (breadcrumbs)

Click the Websites link at the top to return to the Websites page:

Hosting plan informaton

The URL for your website as well as they type of hosting plan is listed at the top:

Hover over your website's URL (anicon appears next to it) and click it to open the selected website. The website URL also turns blue on hover.

Website and Hosting tabs

The Manage Websites page also includes the Website and Hosting tabs, where you can make further adjustments to your website:

Click either one to view specific settings and features, which are described in the following sections.

Website tab

Depending on the type of hosting plan you have and the features that are enabled (or disabled) on it, some of the following features may not appear in your panel.

The following items appear on the Websites tab for you website.

Add Hosting

If your website does not have a hosting plan, the following appears at the top:

Click the blue Add Hosting button, which opens the Manage Domains page for your site.

Manage Your Site

The Manage Your Site box includes information about how to access your server files using your own third-party FTP client or DreamHost's WebFTP:

  • Click the Show Login Info link to view the setting for your server, including host name, username, and password, as well as a link to the Manage Users page.
  • Click the Manage button to open WebFTP.

WordPress

If WordPress is installed on your website, the following box appears where you can open the wp-admin page or uninstall WordPress. You can also obtain a migration key that allows you to use DreamHost's automated migration plugin:

The following box appears if WordPress is not installed yet on your website, which allows you to quickly install WordPress by clicking the Install button :

PHP

The PHP box displays the current PHP version on your website:

Click the Modify arrow, which expands the box where you can make adjustments to the PHP version for your website.

Domain

Shows the status of your domain registration (including the renewal date), and DNS records:

Security

Shows whether an SSL certificate is acitve on your domain and if DreamSheild is enabled:

Email

Shows type of email plan and the number of email addresses set up on your domain:
