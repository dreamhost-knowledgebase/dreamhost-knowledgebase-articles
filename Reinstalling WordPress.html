Overview

This process reinstalling the core files in the /wp-admin and /wp-includes directories and will not affect your website.

Reinstalling in the WordPress dashboard

  1. Log into your WordPress site at example.com/wp-login.php.
  2. On the top left, hover over Dashboard. A menu pops out to the right. Click Updates.
  3. On the next page, click Re-install Now.

Reinstalling using FTP

If you don't have access to your WordPress dashboard, you can upload the files using an FTP client.

  1. Visit https://wordpress.org/download/ to download WordPress to your computer.
  2. Unzip the file on your computer.
  3. Log into your site using an FTP client.
  4. Upload only the /wp-admin and /wp-includes directories. When prompted, choose to overwrite those files.

Reinstalling using wp-cli

wp-cli is a command line tool you can use to manage your WordPress site.

  1. Log into your site via SSH.
  2. Run ls -la to ensure you're in your site's directory. You'll see a list of all of your WordPress files.
  3. Run the following command to download your core files. 
    [server]$ wp core download --skip-content --force
