The ‘Goodies’ menu contains links for other parts of the DreamHost panel that offer additions to your website. These can be used to optimize your overall experience.

MySQL Databases

The (Panel > ‘Goodies’ > ‘MySQL Databases’) page is where you can add/manage a database which may be used with your website.

View the following article for details:

  • MySQL overview

One-Click Installs

The (Panel > ‘Goodies’ > ‘One-Click Installs’) page is where you would install and manage one-click applications such as MediaWiki, WordPress, concrete5, and Joomla.

View the following article for further details:

  • What One-Click Installs does DreamHost offer?

Subversion

The (Panel > ‘Goodies’ > ‘Subversion’) page is where you can add/manage a subversion repository. Subversion is a "version control system" to help you manage projects ("repositories") that include many files and/or contributors.

View the following article for details:

  • Subversion overview

Cron Jobs

The (Panel > ‘Goodies’ > ‘Cron Jobs’) page is where you can add/manage a cron job. Cron is a way of performing scheduled tasks on UNIX systems. Each entry in a user's "crontab" file is called a "cronjob."

View the following article for details:

  • Crontab overview

Jabber IM

On the (Panel > ‘Goodies’ > ‘Jabber IM’) page, you can add Jabber users to communicate via instant message.

Jabber is an open-source instant messaging platform that uses open, XML-based protocols to create the standard functionality people expect of an IM system: one-to-one chat, multi-user chat, the ability to subscribe to someone else's presence, and so on. The great thing about Jabber on DreamHost is that you can host your OWN Jabber chat server on your domain.

View the following article for details:

  • Jabber overview

Htaccess/WebDAV

The (Panel > ‘Goodies’ > ‘Htaccess/WebDAV’) page is where you can enable password protection or WebDAV.

You can use this web interface to create usernames and passwords for your friends and associates to access password-protected sections of your websites. You can even use an .htaccess file to help prevent hotlinking of your website files by outside parties.

View the following articles for details:

  • .htaccess overview
  • WebDAV overview

Block Spiders

On the (Panel > ‘Goodies’ > ‘Block Spiders’) page, you can create an .htaccess file to block specific search engines.
