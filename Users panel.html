Overview

The user section of the panel provides access in order to add/manage users, add/edit SSH Keys, and create account privileges to give specific users selected access to your panel. You can also configure specialized Unix groups for advanced user control.

Manage Users

The Manage Users page is where you can add a new FTP/SFTP/Shell user to your server.

View the following pages for further details:

  • Enhanced User Security
  • Types of DreamHost users

SSH Keys

On the SSH Keys page, you can add and edit SSH Keys for your server. SSH keys can be used for passwordless access to your VPS.

Account Privileges

On the Account Privileges page, you can add specific email addresses and decide exactly what they can access in your panel. This gives you the ability to provide selected access to your panel.

View the following article for details:

  • Account Privileges

Unix Groups

The Unix Groups page is for advanced users where you can create Unix groups on the server which allow different Shell users to access specific directories under another user.

View the following article for details:

  • Unix Groups
