You can soft or hard reboot a running instance using the OpenStack command line client. A soft reboot attempts a graceful shut down and restart of the instance. A hard reboot power cycles the instance.

In the following examples, myInstance is the name of your instance.

You can view the name of your instance in the DreamCompute dashboard.

By default, when you reboot an instance, it is a soft reboot.

[user@localhost]$ openstack server reboot myInstance

[user@localhost]$ openstack server reboot --hard myInstance
