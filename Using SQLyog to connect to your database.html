Overview

The following describes how to set up WebYog's free trial version of SQLyog to connect to your DreamHost MySQL database.

Connecting to your database using SQLyog

  1. Download and install SQLyog.
    The 'Connect to MySQL Host' dialog box appears when you open SQLyog:
  2. Click the New button.
    The 'New Connection' dialog box opens:
  3. Enter a name for the new connection and click the OK button. You can name it anything you wish.
  4. Enter the following information under the MySQL tab:
    • MySQL Host Address — The MySQL host address can be found on the MySQL Databases page in your panel. For more information, see the Finding your database login credentials article.
    • Username — The username for the database.
    • Password — The password for the database.
  5. Enter the following information under the SSH tab:
    • SSH Host Address — The SSH host address can be found on the Manage Account page in your panel.
      Scroll down to the Other Active Services section and click the View Services button to the right.
      Under the Users section, the SSH/Shell server name appears under the Server column. For more information, see the SSH overview article.
    • Username — The username can be found on the Manage Domains page in your panel, to the right of your domain under the Web Hosting column.
    • SSH Port — Enter '22' as the SSH Port value.
    • Password — The password for the user. You can visit the FTP Users & Files page in your panel to change your password if needed.
  6. When finished, click the Test Connection button.

Once you're able to connect, click the Connect button to connect to your DreamHost MySQL database.
