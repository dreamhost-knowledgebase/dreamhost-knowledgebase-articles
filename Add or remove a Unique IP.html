Overview

You can add a Unique IP to your site in the panel.

This article explains how to add an IPv4 address. If you would rather add an IPv6 address, view the following article:

  • Adding an IPv6 address

Additionally, dreamhosters.com subdomains and sites on DreamPress cannot add Unique IP addresses.

Adding an IPv4 address

  1. Navigate to the Manage Websites page.
  2. Click the Manage button to open the Manage Websites page, which allows you to adjust various settings for your site.
    • If in Grid view, click the Manage button at the bottom of the website box.
    • If in List view, click the Manage button at the far right of your domain name.
  3. Select the Hosting tab.
  4. Scroll down to the Unique IP section and click the Manage button.
  5. Select a payment option.
  6. Click the Add a Unique IP now button.

Please be aware of the following when you add a Unique IP address:

  • Once the IP address has been added, it will take several hours to resolve online.
  • This Unique IP address will automatically renew at the end of its term (monthly or yearly). You'll need to remove it manually to stop any future charges.

Removing a Unique IP address from a domain

To delete a Unique IP address:

  1. Navigate to the Manage Websites page.
  2. Click the Manage button to open the Manage Websites page, which allows you to adjust various settings for your site.
    • If in Grid view, click the Manage button at the bottom of the website box.
    • If in List view, click the Manage button at the far right of your domain name.
  3. Select the Hosting tab.
  4. Scroll down to the Unique IP section and click the Cancel Unique IP button.
  5. Click the Yes, remove button to confirm the removal.
