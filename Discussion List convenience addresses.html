Overview

Convenience Addresses are email addresses that can be created so your list users can use listname@example.com for the list, rather than listname@lists.example.com. Thus, the lists portion is removed in convenience addresses.

Examples of convenience addresses

The following convenience addresses are created when adding a Discussion List named mylist to a domain of example.com.

All addresses in this first column forward to the corresponding Discussion List email address:

Convenience address Discussion List address
mylist-admin@example.com mylist-admin@lists.example.com
mylist-bounces mylist-bounces@lists.example.com
mylist-confirm@example.com mylist-confirm@lists.example.com
