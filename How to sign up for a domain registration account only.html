Overview

The following describes how to create a new DreamHost account that does not include any hosting plans or services. This can be a helpful option in certain situations, such as when you only need to transfer an existing domain to DreamHost.

Setting up a new DreamHost account

To set up a new DreamHost account without any plans:

  1. Navigate to https://signup.dreamhost.com/domain/email/.
  2. Enter your email address and create a password.
  3. Click the Continue button when finished.
    The Register Domain page opens.
  4. Search for a domain or enter one to transfer to DreamHost.
  5. Proceed through the following steps to complete the setup of your domain-registration only account.

When finished, your panel is set up, and you can manage the details for your domain registration.
