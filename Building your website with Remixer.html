The following describes how to quickly build your site in Remixer.

Logging into Remixer

There are two ways to log into Remixer:

  • Log in directly with your DreamHost panel credentials at https://dreamhostremixer.com, or
  • From within the DreamHost panel:
After you log in, the following prompt appears:
  • Click the blue Grant Access button to open Remixer.

Selecting a Theme

Watch the YouTube here:

  • Remixer: Edit a Theme

After you grant access, the main Remixer page opens where you are prompted to “Select a theme to start”. You can change the look and feel of each section by placing your mouse over any of the listed themes.

Within each theme, you have the option to either “Preview Theme” or “Select Theme”.

Preview Theme

When you click the 'Preview Theme' link within a theme, a page opens that shows you the theme in a desktop, tablet, or mobile version.

Watch the YouTube here:

  • Remixer: Preview a theme

Select Theme

When you click the 'Select Theme' link within a theme, a page opens that allows you to edit the various sections within the theme.

Themes are divided into several sections, which are called "Chords." Each Chord has a specific function.

After selecting the theme you wish to use, you are then able to edit that virtually any item within the theme itself.

Editing Text

When you select text, the selection area turns to a blue highlight which you can click to update.

Adding Chords

Each Chord in the editor has the following options:

  • Remix — Change the format of any existing data in a particular Chord
  • New — Add a new Chord to the Theme you have selected
