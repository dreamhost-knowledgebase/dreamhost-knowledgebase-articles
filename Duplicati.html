Duplicati is a free backup client that securely stores encrypted, incremental, compressed backups on cloud-storage services and remote-file servers. It works with Linux, Mac, and Windows.

Duplicati has built-in AES-256 encryption and backups can be signed using GNU Privacy Guard. A built-in scheduler ensures that backups are always up-to-date. The Duplicati project was inspired by duplicity, and while similar, they are not compatible.

This article walks you through how to configure Duplicati with your DreamObjects account.

Configuring Duplicati

  1. Download Duplicati and open the program.
  2. On the left, click 'Add backup'.
  3. Select the option to 'Configure a new backup'.
  4. Give it a name and select AES-256 encryption, built-in.
  5. Enter a strong passphrase you'll remember. 
  6. There are several fields on the next page. Make sure to enter them as follows:
    • Storage Type: S3 Compatible
    • Use SSL: Check this box
    • Server: DreamHost
    • Bucket name: Your bucket name.
    • Bucket create region: Find the location of your DreamObjects server on the (Panel > 'Support' > 'Data Centers') page.
    • Storage class: Leave as default
    • Folder path: Enter a specific folder path if necessary
    • AWS Access ID: Your user's Access Key
    • AWS Access Key: Your user's Secret Key
  7. Click the 'Advanced options' link. More options display:
    • s3-server-name: Manually type in your servername. For example, if it's on the west coast enter: objects-us-west-1.dream.io
  8. Click the 'Test connection' button to confirm you are connecting to DreamObjects.
    You'll receive a message about the username not being correct. Click 'NO'. Then you should see a 'Connection worked!' popup.
  9.  Click the 'Next' button.
  10. On the next page, choose which folders you'd like to backup.
  11. Set a schedule for your backups.
  12. Upload volume size default to 50MB.
  13. Under 'Advanced options' select 'AES-256 encryption, built-in'.
  14. Click the 'Save' button.
  15. Click 'Run now' to backup your files.
