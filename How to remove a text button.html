Remixer includes a feature that allows you to remove unwanted text buttons from most of the web sections.

Removing a text button

To remove a text button:

  1. Open the Editor and select a web section.
  2. Highlight the entire text and any spaces within the text button, and then press the backspace key.
    The button shrinks after pressing the backspace key and the button is marked for removal when your next Remixer project publishes:
  3. After you re-publish, refresh your Editor to ensure the text button is removed from your project:

    After you remove a text button, it is not possible to add it back to the existing web section. Instead, create a new web section and then delete the previous web section.
