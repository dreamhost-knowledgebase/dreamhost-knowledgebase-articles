DreamHost supports a wide variety of technologies to help support and customize your managed hosting environment on shared hosting, VPS, DreamPress, and (to a limited degree) dedicated servers. If you do not find your technology listed in the tables below, please contact DreamHost support to confirm if it's supported.

Apache mods

Mods Supported? Details
mod_rewrite YES Enabled by default on all servers.
mod_perl NO Visit the Perl article for details.
mod_php NO Not supported on Shared hosting, but you may enable this on a VPS or Dedicated server. If sudo access is required, you must install on a Dedicated server using an admin user.
mod_proxy NO Not supported on Shared hosting, but you may enable this on a VPS or Dedicated server. If sudo access is required, you must install on a Dedicated server using an admin user.
mod_security YES This is enabled by default. Visit the Mod Security article for details.
