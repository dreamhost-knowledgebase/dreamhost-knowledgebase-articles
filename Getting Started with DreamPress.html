1. Using the DreamPress dashboard

If you're new to WordPress or DreamPress, DreamHost wants to make sure everything is easy to find. There are two separate locations to manage your website and DreamPress hosting experience:

  • the WordPress Admin Dashboard
  • the DreamPress Dashboard

The WordPress Admin Dashboard

The WordPress Admin Dashboard (wp-admin) is where you can add and edit content and features on your WordPress website.

  • How to login to your WordPress Admin Dashboard

The DreamPress Dashboard

The DreamPress Dashboard is where you can manage hosting details for your website. This includes access to the server files and database that power your WordPress site, and added features such as easy backups and restores, as well as an automatic login to your wp-admin to edit your content. You can also access general DreamHost account management such as domain registrations (purchasing new domain names), billing, and additional services if you want more space or websites.

You can log in to your DreamPress Dashboard directly by clicking this DreamPress button.
