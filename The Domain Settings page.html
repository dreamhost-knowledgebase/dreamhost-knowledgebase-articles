Overview

The Manage Websites page is where you access the Domain Settings page for any of your websites.

This page provides a streamlined view of all of your domains, making it easy to make setting adjustments, configure hosting plans, and configure DNS.

Grid or List view

You can view your websites in either Grid or List View. The following describes each display setting.

Grid View 

Grid View shows basic information for all your domains in a graphical grid format (the screenshot of your site(s) updates every 24 hours):

List View 

List View shows all of your domains in a list-style view, which is ideal if you have many domains to display:

Opening the Domain Settings page

To open the domain settings page:

  1. Navigate to the Manage Websites page.
  2. Click the Manage button to open the Manage Websites page, which allows you to adjust various settings for your site.
    • If in Grid view, click the Manage button at the bottom of the website box.
    • If in List view, click the Manage button at the far right of your domain name.
The domain's settings page opens.

The following tabs appear at the top of the page:

  • Website
  • Hosting
  • Staging
  • DNS

Website

The Website tab is where you can manage various aspects of your website configuration, such as:

  • Website and database files
  • WordPress configuration
  • PHP version
  • Security
  • Email
  • SEO
  • Web directory location
  • Passenger

Hosting

The Hosting tab displays your website's hosting plan, billing date, and server name. This page allows you to adjust the following:

  • Hosting plan
  • Add a Unique IP
  • Restore website files
  • Redirect, Mirror, or Park your domain
  • Delete the domain

Staging

The Staging tab offers you the ability to create a staging website by upgrading to a DreamPress plan. 

Staging websites allow you to test new plugins, themes, features, or updates to your WordPress website without it affecting the live version.
