As part of an effort to better maintain and support One Click Install applications, DreamHost is no longer able to provide new installations or upgrades for XODA through the DreamHost panel. The following provides information on how to maintain and where to find further information on upgrading to the latest version.

How do I continue to maintain XODA?

In order to ensure your XODA installation remains secure and compatible with the latest versions of PHP, DreamHost strongly recommends bookmarking and regularly checking the official XODA site, here.

How can I find the latest version of XODA?

The last version offered by DreamHost was 0.6.3, and the latest version available from the developer is 0.7.0 (which you can download here). It is advisable to upgrade as soon as possible using the instructions from XODA, here.
