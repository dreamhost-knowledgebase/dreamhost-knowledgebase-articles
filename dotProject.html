dotProject is a project management tool that is designed to assist in the management of defined project work. It offers project managers a series of tools to manage project tasks and simplifies the sharing of content between managers and members.

You can visit dotProject's website for more details and to see examples of it being used.

http://dotproject.net/

dotProject is not compatible with PHP 7. If you wish to use this software, make sure your site is using PHP 5.6.
