FAQs

What is Remixer?

Remixer is DreamHost's new click-to-edit site builder tool!

You can easily create a ready-to-publish website by following these simple steps:

  1. Select a theme
  2. Edit a theme
  3. Preview your page
  4. Assign the page to your domain

The Remixer setup articles describe the entire process from picking a theme to publishing your assets to a DreamHost domain.

How do I start using Remixer?

  1. Open https://dreamhostremixer.com/.

    Google Chrome browser is preferred.

  2. Enter the same account owner login credentials you use for the DreamHost panel.
  3. When the 'Authorize Remixer...' prompt appears, click the blue Grant Access button to open Remixer.

Do I need to know HTML, JavaScript, or CSS?

No. Remixer is entirely template-based, and requires no hand coding whatsoever. You can create a fully-functional website in minutes using an easy-to-use, click-to-edit site builder.

Can I access the HTML files directly? Does Remixer have an HTML editor to manually modify the theme or chord?

View the following article for an overview of 'Chords'.

  • Remixer Chords

At this time, Remixer does not have an HTML editor built in or any method to access the actual HTML files.

Remixer themes and chords have been specifically designed to look and fit a certain way, and DreamHost recommends you use the Remix option under each chord if you wish to try a different chord style.

Do I need anything else?

No. All you need is a computer/browser and the ability to point, click, drag, and drop!

DreamHost recommends using a desktop computer with Google Chrome browser for Remixer.

How do I get support for Remixer?

You can submit a regular support ticket (Panel > 'Support' > 'Contact Support') or email Contact Remixer Support during regular staffing hours, 10AM to 6PM Pacific Time, Monday through Friday.

Remixer support is not currently available over Live Chat.

My domain is showing a 'Page Not Found' site

Review the following instructions on how to publish to your domain name:

  • How do I publish my first Remixer page to a domain?

I just published, but my domain isn’t loading the site yet

Double-check to make sure you have assigned the domain or sub-domain to your page on the Pages menu.

DNS changes are required if you are publishing to your domain or sub-domain on Remixer for the first time.

  • If your domain’s nameservers are pointed to DreamHost, it automatically updates for you within 24 to 48 hours.
  • If your domain’s nameservers are not pointed to DreamHost, you must update the DNS records A records at the company that hosts your nameservers. You would point them to your site's A records.

Remixer requires that your DreamHost Panel web hosting for the domain is set to “DNS Only” prior to publishing. To set up DNS only:

  1. Open the (Panel > ‘Domains’ > ‘Manage Domains’) page.
  2. Click the 'Remove' link under the Web Hosting column.
  3. Return back to Remixer and republish your site from Pages.

Your site should be visible after a few hours of DNS propagation.
