The following instructions are only if you have a Shared hosting plan in addition to a VPS, Dedicated, or DreamPress plan.

Viewing your hosting plan type

You can view all services billed to your account on the (Panel > 'Billing & Account' > 'Manage Account') page. Here you'll see a list of all services you currently use.

You must remove all users from the Shared server in order to proceed. First, make sure you backup any important files under the Shared user you wish to keep. View the following articles for information on how to backup your user data:

  • How do I backup my entire account
  • How can I manually backup my data?

Once your data has been backed up, navigate to the (Panel > 'Users' > 'Manage Users') page. To the right of any user (under the 'Machine' column) you'll see the name of the server. For example, 'flower', 'gervais', 'tool' etc. Your VPS will instead look like ps123456.

If it has a name, it is a Shared server.

Click the Delete button to the right of the user to permanently delete it.
