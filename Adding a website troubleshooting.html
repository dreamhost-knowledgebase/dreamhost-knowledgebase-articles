Overview

It's possible you may encounter an error when adding a website and hosting at DreamHost. This article lists a few common errors and how to resolve them.

'example.com is almost here!' message on my website

A newly hosted domain includes a file named quickstart.html in its root directory. This file contains a placeholder page that prompts you to upload your site, as follows:

DreamHost servers are configured to load this file by default unless you delete it or upload your own website files containing your own index file.

Error messages when adding a domain

You must add the parent domain

You can't add that domain: 
you must add the parent domain example.com to your account before adding a subdomain to it.

This error means that the primary domain name (example.com) is not yet added to your account. A subdomain (blog.example.com) can only be added if its primary domain is created first. The primary domain must be added first even if you don't plan on hosting the second-level domain here at DreamHost.

You don't have permission to add that domain

You can't add that domain: you don't have permission to add.

This error means that the domain you are trying to add to your account is already hosted on another DreamHost account. If you are moving the domain from an existing account that you have access to, you must remove the domain from the old account first before you add it to a new account. If you do not know what account the domain is currently hosted on, please contact support.

The domain name already exists in the system

The domain name already exists in the system.

If you see this error message, please confirm that the domain is not hosted on another DreamHost account you have access to. If you do not know what account the domain is currently hosted on, please contact support.
