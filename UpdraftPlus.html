Overview

UpdraftPlus allows you to easily backup your WordPress site to DreamObjects.

Installing the plugin

  1. Log into your WordPress site.
  2. In the left menu, choose ‘Plugins > Add New’.
  3. Search for ‘UpdraftPlus’. Once found click the Install Now button.
  4. Back on the 'Plugins' page, make sure to click 'Activate' under the plugin.

Configuring the plugin

  1. Make sure you're on the plugin page.
  2. Click the Settings link under the plugin.
  3. On the main page for the plugin, click the Settings tab.
  4. Configure UpdraftPlus to backup your files and database at the intervals you prefer.
  5. Under the Remote Storage section choose DreamObjects.
  6. Enter the following:
    • DreamObjects access key: View the DreamObjects Keys article for instructions on how to locate this.
    • DreamObjects secret key: View the DreamObjects Keys article for instructions on how to locate this.
    • DreamObjects location: Enter the name of the bucket you'd like to backup to.
    • DreamObjects end-point: Choose 'objects-us-east-1.dream.io'
  7. Click the Test DreamObjects Settings button.
    A 'Success' message appears.
  8. Scroll down and click the Save Changes button.

Creating a manual backup

At this point, your plugin is set to backup at the time interval you specified above. However, you can always backup manually anytime you like.

  1. Open the plugin.
  2. Click the Backup / Restore tab.
  3. Click Backup Now.
    A status bar displays as your backup processes. Once complete, you'll see it in the Existing backups section.

Restore a backup

  1. Open the plugin.
  2. Go to the Existing backups section.
  3. Click the Restore button to the right of the backup you wish to restore.
