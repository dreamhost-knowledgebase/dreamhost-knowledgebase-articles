Overview

The user section of the panel provides access in order to add/manage users, add/edit SSH Keys, and create account privileges to give specific users selected access to your panel.

Manage Users

The FTP & SSH Users page is where you can add a new FTP/SFTP/Shell user to your server.

View the following article for further details:

  • Types of DreamHost users

SSH Keys

On the SSH Keys page, you can view the SSH Fingerprints assigned to your server. You can then verify you're connecting to the correct server via SSH when it prompts you to save the Fingerprint and SSH Key.
