Below are some applications users have written that use the DreamHost API.

Desktop Apps

  • https://sourceforge.net/projects/dreamhostadmin/ — Windows desktop application for remote account management

Linux command line

  • https://deekoo.net/technocracy/lsdh/ — Interactive command line interface
  • https://github.com/helderco/Dreamhost-Full-Auto-Backup/ — Script that backs up all your MySQL databases and Shell users into your backup account

Announcement List management

  • domador.net/extras/dalmatian/ — Windows application for managing Announcement Lists
  • https://terrywysocki.com/ckeditor_announcementlist_api.php — Web based WYSIGYG editor via API for Announcement List posts

Programming libraries

  • WWW::DreamHost::API — Perl interface to the API
  • Another PHP Class — Another PHP class
  • Fog — Supports DreamHost DNS API. See the examples.

DNS

  • https://github.com/gsiametis/dreampy_dns/ — Another Python dynamic DNS script with IPv6 support
  • https://github.com/clempaul/dreamhost-dynamic-dns — Dynamic DNS bash script
  • https://github.com/Erutan409/dreamhost-api-powershell — PowerShell Cmdlet Functionality
