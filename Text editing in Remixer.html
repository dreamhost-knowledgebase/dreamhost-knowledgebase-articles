Overview

When you are editing a page, you can edit and style text however you like. The built-in editor allows you to do several things such as:

  • align text
  • choose font
  • change size
  • bold/italic
  • link text
  • add lists

You can only edit/add text within a predefined text field section of a 'Web Section'. It's not possible to add text outside of a 'Web Section' or remove a text box from a 'Web Section'.

How to edit text

  1. Make sure on the theme's edit page.
  2. Locate the text to update.
  3. Click on the text. The text is highlighted and the editing toolbar appears.
  4. Edit (or delete) the text as needed. For example, you can change the font size, font style, alignment, or color.
  5. When finished editing, click outside the section to auto-save.
