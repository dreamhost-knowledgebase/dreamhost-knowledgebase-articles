  • align text
  • choose font
  • change size
  • bold/italic
  • link text
  • add lists

You can only edit/add text within a predefined text field section of a 'Web Section'. It's not possible to add text outside of a 'Web Section'.

How to edit text

  1. Make sure on the theme's edit page.
  2. Locate the text to update.
  3. Click on the text. The text is highlighted and the editing toolbar appears.
  4. Edit (or delete) the text as needed.
  5. When finished editing, click outside the section to auto-save.

Linking text (hyperlink)

You can create a hyperlink to link your text to another URL. Highlight the text you want to link. An edit toolbar appears

Click the link icon. Below you can enter the destination of the link.
