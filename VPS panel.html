On the (Panel > 'VPS' > 'Dashboard’) page, you can add a new VPS or manage an existing one.

If you already have a VPS, you can click the Configure button to the right. This opens a new page where you can update the following: 

  • Change the name of the server
  • Set Root SSH Keys (Only available on old accounts with an admin user. New accounts do not allow admin users.)
  • Change webserver from Apache to Nginx
  • Deactivate FTP
  • Activating POP3/IMAP server

For further information on private servers, view the following category:

  • VPS Hosting
