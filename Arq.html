Arq is a Mac and Windows app that allows you to make online backups that support DreamObjects. Arq backups are encrypted with a password before they’re uploaded so your data is protected.

The installation and configuration of Arq varies depending on your operating system. Click a operating system below to jump to the proper instructions.

  • Windows
  • Mac

Connecting with Arq on Windows

  1. Launch Arq.
    When launching Arq for the first time, you are prompted to choose your backup destination type:
  2. Click the ‘Other S3-Compatible Service’ option, and then click the Continue button.
  3. Enter the following:
    • S3-Compatible Server URL: Enter the hostname https://objects-us-west-1.dream.io.
    • Access Key ID: Found in your DreamHost Panel. See the keys article for more information.
    • Secret Access Key: Found in your DreamHost Panel. See the keys article for more information.
    • Request Signature Version: Ensure that the signature version defaults to ‘Signature Version 2’.
  4. When finished, click the Add Destination button to continue.
    Arq prompts you to enter a new bucket name, or select an existing bucket:
  5. Create a bucket (and enter a name) or select an existing bucket, then click the Add button to finish.
    After your bucket is configured, Arq prompts you to set up a new backup schedule or to restore files from this bucket (which requires existing backups in your bucket):
  6. Click the Set Up Backups button.
  7. Enter an encrypted password which provides additional security for the backup that Arq creates.
  8. After you enter an encrypted password, be sure to write down the password you created and store it somewhere safe.
  9. Choose the folders to backup.
    • On Windows systems, Arq defaults to backing up the entire C: drive excluding only the Recycle Bin and the pagefile.sys file.
  10. To add or remove files or folders, click the Edit backup suggestions button, which allows you to customize the selection.

Arq is now configured to backup your data.

Connecting with Arq on Mac

  1. Launch Arq.
    When launching Arq for the first time, you are prompted to choose your backup destination type:
  2. Enter the following:
    • Choose your backup destination: Select ‘DreamObjects’ from the drop-down list.
    • Public Key: Found in your DreamHost Panel. See the keys article for more information.
    • Secret Key: Found in your DreamHost Panel. See the keys article for more information.
    • Retrieve DreamObjects Keys — Click to open your DreamHost Panel to find your keys.
  3. Enter a new bucket name or select an existing bucket from the drop down list.
    After your bucket is configured, Arq prompts you to set up a new backup schedule, or to restore files from this bucket (which requires existing backups in your bucket):
  4. Click the Set Up Backups button.
  5. Create an encrypted password for your backup which allows additional security for the backup created.
  6. After you enter an encrypted password, be sure to write down the password you created and store it somewhere safe.
  7. Choose the folders to backup.
    • On Mac systems, Arq defaults to backing up the /Users directory, excluding files it knows are unnecessary like cache and page files.
  8. To add or remove files or folders, click the Edit backup selections... button, which allows you to customize the selection.

Arq is now configured to backup your data.
