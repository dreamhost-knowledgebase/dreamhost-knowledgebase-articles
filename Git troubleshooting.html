gnutls_handshake() failed: Handshake failed

You may see this error when connecting to certain repositories from an older DreamHost VPS or Dedicated Server running Ubuntu 14 (Trusty). The error is related to an Ubuntu 14 bug. The following explains what servers this may occur on.

Shared

DreamHost has already updated Shared web servers to Ubuntu Bionic 18 where you will no longer run into this error.

VPS

VPS plans purchased before March 2020 run Ubuntu 14.

VPS plans purchased after March 9th, 2020 run the operating system Debian Stretch where you will no longer run into this error.

Dedicated

Any newly purchased Dedicated Server runs Bionic. DreamHost is currently upgrading old Dedicated Servers to Bionic where you will no longer run into this error.
