Overview

DreamHost offers several versions of PHP for you to use. The following article lists which PHP versions are available in the panel:

  • What versions of PHP are available at DreamHost?

There is also an extended support plan available if you require an outdated version. View the following article for further details on this service.

  • PHP Extended Support

Viewing default settings

Each version of PHP has several default settings you can view by creating a phpinfo.php file. This is a simple PHP file that you can add to your website. When you view the page, it displays the default PHP settings configured on your domain. View the following article for steps on how to create this file:

  • Viewing your PHP version and settings

Adjusting default settings

You are able to adjust the default PHP settings by creating a custom phprc file. This is a custom configuration file that allows you to adjust specific values. View the following articles for steps on how to create this file via FTP or SSH:

  • How do I create a phprc file via FTP?
  • How do I create a phprc file via SSH?
