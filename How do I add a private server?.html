Adding the VPS

The steps to add a new VPS are different depending on if you are adding a VPS for the first time or not.

The instructions below explain how to add a VPS for the first time, or add an additional VPS to your account.

Moving a site to a VPS does not affect your email service. Email is on a different server and will continue to function normally.

Adding a VPS for the first time

  1. Go to the Dashboard page.
    The ‘Managed VPS Hosting with SSDs’ page opens:
  2. Click the blue Add VPS button near the bottom of the page.
  3. Select one of the first two radio buttons to migrate all data from your Shared server to your VPS, or to not migrate any data.

    You can always migrate data in the future.

  4. Click the blue Get This VPS button for the plan you wish to purchase.
    The window changes to display a 'Monthly' or 'Yearly' option:
  5. Click the 'Monthly' or 'Yearly' option to purchase the VPS.
    A success message displays confirming the VPS has been purchased:

An email is sent to you after the VPS is provisioned. This usually completes in a few hours. 

Adding an additional VPS to your account

  1. Go to the Dashboard page.
  2. Click the Add VPS button.
    A list of tiered plan options appears:
  3. Click the blue Get This VPS button for the plan you wish to purchase.
    The window changes to display a 'Monthly' or 'Yearly' option:
  4. Click the 'Monthly' or 'Yearly' option to purchase the VPS.
    A success message displays confirming the VPS has been purchased:

An email is sent to you after the VPS is provisioned. This usually completes in a few hours.
