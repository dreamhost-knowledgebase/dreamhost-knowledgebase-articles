DreamCompute utilizes SSH keys for accessing your instances, instead of using passwords alone. This is done for increased security and simplifies managing users for OpenStack since the public key is injected into the instance at boot. This setup prevents the need to modify files or passwords on the operating system on startup.

This article shows different ways to create a Key pair. You only need to choose one option below.

  • Create a new Key pair in the DreamCompute dashboard
  • Create a new Key pair in Windows
  • Create a new Key pair in Mac or Linux

SSH key Components

There are two parts to an SSH key:

  • a private key
  • a public key

The private key is known only to you and should be safeguarded. On UNIX-based systems such as Mac and Linux, the private key is generally stored in:

/home/myuser/.ssh/id_rsa

The public key can be shared with any server would like to allow ssh access to. On the server, the public key is stored in:

/home/myuser/.ssh/id_rsa.pub

SSH keys generated on Windows systems using the PuTTY key generator (puttygen.exe) are not stored in a uniform location, but instead wherever the user chooses to save them.

Creating a new Key pair in the DreamCompute dashboard

The easiest way to generate a key is to have DreamCompute generate one for you.

  1. Navigate to the (Panel > 'Cloud Services' > 'DreamCompute') page.
  2. Click the 'View Dashboard' button.
  3. On the left, click 'Compute'.
  4. From the dropdown click 'Access & Security'.
  5. Click the 'Key Pairs' tab.
  6. On the top right click the 'Create Key Pair' button.
  7. Give the key a name, and click 'Create Key Pair'.
  8. Your browser will start downloading the private key named the same as they name given with a .pem file extension.

Creating a new Key pair in Windows using puttygen.exe

This program can be downloaded from the PuTTY website. View the 'How do I configure PuTTY' article for instructions on how to download and where to save the files on your computer.

  1. Navigate to the directory you saved the puttygen file and open it. The 'PuTTY Key Generator' window displays.
  2. On the bottom right, click the 'Generate' button.
  3. Follow the instructions and move your mouse in the blank area to generate random data.
  4. Copy and paste the text at the top labeled 'Public key for pasting into OpenSSH authorized_keys file' into a blank notepad file. This will be imported into the DreamCompute dashboard.
  5. Click the 'Save public key' button to save a modified version of the public key. Please note that this public key file will not be imported into the DreamCompute dashboard.
  6. Click the 'Save private key' button to save the .ppk (PuTTY Private Key File) file for use with PuTTY later.
  7. Finally, import the public key into your DreamCompute dashboard. 

Creating a new Key pair in Mac OS X or Linux using ssh-keygen

This program can be found on Mac, Linux and other UNIX-based operating systems.

On your home computer:

  1. Generate an RSA private key using ssh-keygen (unless you have already created one).
    • If you’re using Linux or Mac OS X, open your terminal and run the following command under your username:
    [server]$ ssh-keygen -t rsa
    This creates a public/private keypair of the type (-t) rsa.
    Generating a public/private rsa key pair.
Enter the file in which you wish to save they key (i.e., /home/username/.ssh/id_rsa).
    Once the keypair is created, you are prompted to enter the following items.
  2. Click Enter on your keyboard to continue. 
    Enter a passphrase (leave empty for no passphrase).
  3. Click Enter on your keyboard to continue. 
    Enter same passphrase again:
  4. Click Enter on your keyboard to continue.
  5. When finished, click Enter on your keyboard.
    The following message appears:
    Your identification has been saved in /home/username/.ssh/id_rsa
Your public key has been saved in /home/username/.ssh/id_sra.pub

The key fingerprint is:
ar:bc:d3:9e:g3:1f:63:6f:6b:32:2e:97:ee:42:e1:be username@server.dreamhost.com

The key’s randomart image is:

  6. Finally, import the public key into your DreamCompute dashboard.

Checking For An Existing Key

For a Windows system, there isn’t an easy way to find where keys were generated or stored. A private key generated from puttygen.exe is saved with the .ppk file extension, but the other files it generates has no default extension. You could search your system for .ppk files and perhaps find your public key in the same location.

On Mac and Linux systems, you can check in the above described location:

[user@localhost] ls ~/.ssh/id_rsa*
/home/myuser/.ssh/id_rsa  /home/myuser/.ssh/id_rsa.pub

Import an existing SSH key into the DreamCompute dashboard

  1. Find the public key file and open it to copy its contents.
  2. Navigate to the (Panel > 'Cloud Services' > 'DreamCompute') page.
  3. Click the 'View Dashboard' button.
  4. On the left, click 'Compute'.
  5. From the dropdown click 'Access & Security'.
  6. Click the 'Key Pairs' tab.
  7. To the right, click the 'Import Key Pair' button.
  8. Give it a name and paste in the contents of your public key.
  9. Click the 'Import Key Pair' button to save it.
  10. You can now see it in the dashboard.
