Overview

When you first add a domain to your panel as Fully Hosted, you must assign it to a web directory. This is the folder on the server where the domain’s files are located.

Generally, you should never need to change this directory. Only certain custom configurations will require you to do so. This article shows you how to change the web directory of your domain.

Changing your domain's web directory

  1. Navigate to the Manage Websites page.
  2. Click the Manage button to open the Domain Settings page, which allows you to adjust various settings for your site.
    • If in Grid view, click the Manage button at the bottom of the website box.
    • If in List view, click the Manage button at the far right of your domain name.
  3. Scroll down to the Additional Settings section.
  4. Click the Modify button to the right of the Paths subsection.
  5. Enter the new directory on the server from which you want to load this domain. For example, in the text field that appears, enter: example.com/blog.

    Please note that this field is case-sensitive, and the folder needs to already exist on the server under your domain’s user.

  6. Click the Save Changes button.
