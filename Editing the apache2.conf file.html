Overview

To edit the apache2.conf file, you must have a Dedicated server with an admin user.

root/sudo access is not available on VPS machines.

Location of the apache2.conf file

If your Dedicated server is running Ubuntu 18 (Bionic) or Debian Buster, you can find this file in the following location:

/etc/apache2-apache2-INSTANCE/apache2.conf
