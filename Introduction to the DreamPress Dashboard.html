Overview

The following describes how to use the various features on the DreamPress Dashboard page.

Don’t have DreamPress yet?

You can upgrade an existing WordPress site to DreamPress using the 'Upgrade to DreamPress' link on the Manage Domains page, or go to the Managed WordPress page to purchase.

Accessing the DreamPress Dashboard

To access the DreamPress Dashboard, click the Managed WordPress button in your panel.

When the Managed WordPress page opens, click the Manage button to the right of your DreamPress domain to open the dashboard page for your website.

WP-Admin

The WordPress Admin Dashboard (wp-admin) is where you can add and edit content and features on your WordPress website:

If the Manage WordPress button does not automatically log you into your WordPress Dashboard and instead takes you to a login page, then please Contact Support for assistance.

Jetpack (Plus and Pro plans)

DreamPress Plus and Pro customers have the added benefit of access to a FREE subscription to Jetpack’s Professional level plan. Jetpack Professional users have access to hundreds of themes free of charge with theme support, as well as free real-time offsite backups, unlimited off-site video hosting, Elasticsearch, and more.

See the following article for more information about Jetpack:

  • Activate Jetpack Professional for DreamPress Plus and Pro customers

DreamPress Dashboard features

The following four 'tabs' appear on the dashboard, which allows you to make further adjustments to your DreamPress site:

Each of the tabs (Details, Domain, Backups, and Staging) are described in the following sections.

Details tab

The Details tab shows you information about your current DreamPress plan:

On the Details tab, you can view the following:

  • The current DreamPress plan (as well as a link that allows you to edit or remove a plan)
  • The renewal date
  • How much storage your DreamPress site has used (as well as a link that allows you to increase/decrease storage)

You can also upload files (using WebFTP) and make database adjustments on the Details tab:

File Upload

Click the Manage Files button to automatically log into your WebFTP account to access your website files. Toggle the Show Credentials selector to 'On' to view your FTP credentials. 

See the following article for more information about WebFTP:

  • Using WebFTP

Database (MySQL)

Click the Manage Database button to automatically log into your phpMyAdmin account to access your database files. Toggle the Show Credentials selector to 'On' to view your MySQL credentials. 

See the following article for more information about accessing your DreamPress database:

  • Access your DreamPress database using phpMyAdmin

Domain tab

The Domain tab shows you information about your DreamPress domain:

You can change the domain name on your site (by either selecting a temporary domain or an existing domain). You can also register a new domain. See the following article for more information about changing a DreamPress domain:

  • Changing your domain name on DreamPress

Backups tab

The Backups tab allows you to view and easily restore previous backups, as well a create a new backup:

See the following article for more information about creating and restoring backups:

  • How to create and restore backups

Staging tab

The Staging tab allows you to set up a staging, which allows you to test changes on your DreamPress site before you publish it:

Staging creates a copy of your site in a "sandbox" environment where you can experiment and preview changes without it affecting what your visitors see. When you're ready, push your changes to your live site with a simple click.
