DreamPress Dashboard features

The DreamPress Dashboard is where you can manage hosting details for your website. When you click the Manage button to the right of your DreamPress domain, you can access options such as:

  • Automatic login to your WordPress Admin Dashboard using the Open WP-Admin button
  • Log in to your DreamPress server using secure FTP
  • Access your DreamPress database using phpMyAdmin
  • Activate Jetpack Professional for DreamPress Plus and Pro customers
