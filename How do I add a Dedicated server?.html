Adding a Dedicated server

  1. Navigate to the Dedicated DashboardÂ page.
  2. Click the Add a Dedicated Server button at the top of the page.
    You'll see a few options here that correspond to the chart in the overview article:
  3. Click the desired CPU cores button. The plan options change for each option.
  4. Once you decide which plan you'd like to purchase, click the 'Migrate existing server' dropdown.
  5. Select to either migrate existing data from another server, or to not migrate data at all.
  6. Click the Add Server button to complete your purchase.

Upgrading to a new Dedicated server plan

If you would like to migrate from you current 'Blue Moon' dedicated server to a new 'Summer Moon' plan, log in to your panel and navigate to the Contact Support page.

Create a ticket for technical support requesting that your data on the current 'Blue Moon' server be migrated to a new plan. Please allow at least 3 days for this to take place after support has responded.
