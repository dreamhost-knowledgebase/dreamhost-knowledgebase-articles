Overview

Related article: Websites panel page

The Manage button appears on the Manage Websites page:

  • In Grid view, the Manage button appears at the bottom of the website box.
  • In List view, the Manage button appears at the far right of your domain name.

Click the Manage button to open the Manage Websites page (except for DreamPress and Remixer plans, which instead link to their own manage page).

The following topics are described in this article:

  • Upper display
  • Website tab
  • Hosting tab
  • DNS tab

Upper display

The upper display provides top-line information about your website, navigational tools, and two tabs (Website and Hosting) that provide more detailed information.

Return to the Websites page (breadcrumbs)

Click the Websites link at the top to return to the Manage Websites page.

Hosting plan information

The URL for your website as well as the type of hosting plan is listed at the top.

Hover over your website's URL (anicon appears next to it) and click it to open the selected website. The website URL also turns blue on hover.

Website, Hosting, and DNS tabs

The Manage Websites page also includes the Website, Hosting, and DNS tabs, where you can make further adjustments to your website.

Click a tab to view specific settings and features, which are described in the following sections.

Website tab

Depending on the type of hosting plan you have and the features that are enabled (or disabled) on it, some of the following features may not appear in your panel.

The following items appear on the Websites tab for your website.

Add Hosting

If your website does not have a hosting plan, an information box appears at the top.

Click the blue Add Hosting button, which opens the Hosting Plan page for your site. 

Manage Your Site

The Manage Your Site box includes information about how to access your server files using your own third-party FTP client or DreamHost's file manager.

  • Click the Show Login Info link to view the setting for your server, including hostname, username, and password, as well as a link to the FTP Users & Files page.
  • Click the Manage button to open the file manager.

WordPress

If WordPress is installed on your website, a WordPress box appears.

  • Click the Manage button to open the wp-admin page for your WordPress site.
  • Click the Uninstall WordPress link to uninstall WordPress.
  • Click the Get Key button to obtain a migration key that allows you to use DreamHost's automated migration plugin.

Another box appears if WordPress is not installed yet on your website, which allows you to quickly install WordPress by clicking the Install button.

PHP

The PHP box displays the current PHP version on your website.

Click the Modify arrow, which expands the box where you can make adjustments to the PHP version for your website.

Domain

Shows the status of your domain registration (including the renewal date), and DNS records.

Security

Shows whether an SSL certificate is active on your domain and if DreamShield is enabled.

Email

Shows type of email plan and the number of email addresses set up on your domain.

Hosting tab

Depending on the type of hosting plan you have and the features that are enabled (or disabled) on it, some of the following descriptions may not appear in your panel.

The following items appear on the Websites tab for your website.

Hosting Plan

Shows the current plan, billing date, server name, as well as a link to the Manage Account page where you can change your plan.

Hosting Status

If your website is set to redirect, mirror, parked, or cloaked, the Hosting Status box appears and displays the current status.

Unique IP / Restore Files

You can add a Unique IP or restore your files.

Non-Hosting Options (Redirect, Park, DNS only, etc.)

You can also set up DNS only, redirect, park, and more for your domain.

Delete Domain

You can delete a domain. See the Deleting a domain article for more information
