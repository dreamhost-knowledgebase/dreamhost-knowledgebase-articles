Adding a MySQL VPS

  1. Navigate to the Servers page.
  2. Click the Add Server+ button and select MySQL Virtual Private Server.
  3. Choose a plan and then click the Get This VPS button to purchase it.

    Once added, it may take a few hours to provision the new server.

    • When you sign up for a MySQL VPS, all your databases are moved to it at midnight Pacific time. (The panel shows it as a merge.)
    • No SSH access is available.
  4. When the Cart page opens, review the order summary and then click the Submit Order button.

Deleting a MySQL VPS

If you have a MySQL VPS with 3GB or more of data and no Dedicated server, you cannot use the Delete button in the panel to delete your MySQL VPS. You must decrease the size to less than 3GB for each MySQL VPS database in order to delete it.

If you no longer need your private database server, you can delete it in your panel.

  1. Navigate to the Servers page. You will see your server listed as sqlXXXXX.
  2. To the right of your MySQL VPS, click the three-dot button and select Delete.
  3. Check the box if you wish to transfer all data to another server.
  4. Click the Delete VPS button when finished.

Your data will then be migrated to the shared server within 24hrs.
