Overview

Make sure you are certain you wish to delete a domain. Deleting a domain cannot be undone.

The following describes how to delete a domain using the Websites panel.

Backing up a domain

Before you delete your domain, make sure you back up any data that you want to keep. View the following section for instructions on different methods that describe how to back up your data:

  • Backing up

Deleting a domain

  1. Navigate to the Manage Websites page.
  2. Open the DNS Settings page:
    • If in Grid view, click the 'vertical 3 dots' button under your domain. From the popout menu, click the DNS Settings link.
    • If in List view, click the 'vertical 3 dots' button to the right of your domain. From the popout menu, click the DNS Settings link.
      The domain's DNS records display.
  3. When the Manage page opens, select the Hosting tab.
  4. Scroll down and select the Permanently Delete Domain section.
  5. Click the Proceed with Deletion button to finalize the process.
