Overview

Make sure you are certain you wish to delete a domain. Deleting a domain cannot be undone.

The following describes how to delete a domain using the Websites panel.

Deleting a domain

  1. Navigate to the Websites page.
  2. Click the Manage button for the domain you want to delete.
  3. When the Manage page opens, select the Hosting tab.
  4. Scroll down and select the Permanently Delete Domain section.
  5. Click the Proceed with Deletion button to finalize the process.
