Overview

To display a website on the internet, you need to do two things:

  • Purchase the domain registration
  • Add hosting to the domain

This article assumes you've already purchased the domain registration. 

What is a 'Fully Hosted' domain?

Below are instructions on how to add a domain to your account as 'Fully Hosted'. When a domain is 'Fully Hosted', it's set up on a web server and assigned DNS records. Once DNS have been assigned, you can then visit the website on the Internet. This requires a few steps:

  1. Purchase a domain registration.
  2. Purchase a hosting plan.
  3. Follow the directions below to add the domain to the hosting plan.

Error messages when adding a domain

In some cases, an error message appears when you try to add hosting for a domain on your account. The most common error messages are described below.

You must add the parent domain

If you see:

You can't add that domain: you must add the parent domain example.com to your account before adding a subdomain to it.

it means that the second-level domain is not yet in the system. A subdomain can only be created if its second-level domain is created first (example.com)—even if you don't plan on hosting the second-level domain here at DreamHost.

You don't have permission to add that domain

If you see:

You can't add that domain: you don't have permission to add.

it means that the domain you are trying to add to your account is already hosted on another DreamHost account. If you are moving the domain from an existing account that you have access to, remember to remove the domain from the old account first before you add it to a new account. If you do not know what account the domain is currently hosted on, please contact support.

The domain name already exists in the system

The domain name already exists in the system error can occur in similar situations to the you don't have permission to add error. If you see this error message, please confirm that the domain is not hosted on another DreamHost account and then contact support.
