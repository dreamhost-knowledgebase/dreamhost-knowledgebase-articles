The following improvements and features are added to the new Remixer image editor:

  • Image editor relocated to the left sidebar / better zoom / wysiwyg image placement
  • Drag-and-drop gallery image reordering
  • Quick access to Pixabay search from the image editor
  • Image Filters
  • Other controls

Image editor relocated to sidebar

Previous view:

Challenges

  • Blocks view of website
  • Creates too many scroll boxes and can cut off controls on the image panel as seen above
  • Image as displayed in view box does not always display correctly on website
  • Changing image crops image so you can not go back to original image while editing

New View:

Improvements

  • Image editor no longer covers the website
  • WYSIWYG image editor
  • Easy movement of image with image scale and click to move image in display box
  • What you see is what you get, display box shows exactly what will go on the website
  • Images are moved around, not cropped, so you can always edit freely without an altered image

Drag to Re-order Gallery Images

Previous View:

Challenges

  • Difficult to notice images in the gallery on top of the select image section
  • When deleting images near arrows, there is a clash
  • No drag to re-order images option available, you have to add them in the order you want them displayed

New View:

Improvements

  • Images in gallery listed above main image being edited
  • No button clashing
  • Drag thumbnail images to reorder images on website

Quick Access to Pixabay Search

Improvements

  • Add pixabay search - previously only on Media tab
  • Populate with images that relate to website content (currently learning how easy/difficult this will be)
  • Adding images puts them in your library

Image Filters

Improvements

  • Customers can now choose to put a filter on the picture they are editing
