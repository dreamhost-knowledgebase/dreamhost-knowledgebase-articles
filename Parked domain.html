When adding a domain to your panel, you can configure it several ways:

  • Fully Hosted — This means it appears on the Internet
  • Redirect — This redirects your domain to a Fully Hosted website
  • Mirrored — This mirrors a Fully Hosted website
  • Cloaked — This cloaks your domain and displays a Fully Hosted website
  • DNS Only — This removes hosting and allows you to point your DNS outside of DreamHost

There is a final option called 'Parked'. A Parked domain doesn't do anything and has no settings. All of the options above either display a site, or point to a site to display. A Parked domain is just a placeholder until you decide to do something with the website. 

How to park a domain

When you purchase a domain registration, the domain is automatically added to the Manage Domains page. You can also add a new domain as Parked, or change an existing domain to Parked.

Adding a new domain as Parked

  1. Navigate to the Manage Domains page.
  2. Click Add Hosting to a Domain / Sub-Domain.
  3. Scroll down to the Parked section.
  4. Enter your new domain you wish to park, then click the Park this domain button to enable it.
  5. Back on the 'Manage Domains' page, you'll see it's now set as Parked.

Setting an existing domain as Parked

  1. Navigate to the Manage Domains page.
  2. To the right of your domain, click the 'Edit' link.
  3. Scroll down to the 'Parked' section and click Park this domain.

Once a domain is set to Parked, DreamHost displays a place-holder web site with a generic DreamHost 'coming soon' message.

Parked domains are not hosted at all, so there is no server to log into to upload any data.

How to unpark a domain

To unpark a domain, you just need to assign it a different hosting type. The following are all options you can change to:

  • Fully Hosted — This means it appears on the Internet
  • Redirect — This redirects your domain to a Fully Hosted website
  • Mirrored — This mirrors a Fully Hosted website
  • Cloaked — This cloaks your domain and displays a Fully Hosted website
  • DNS Only — This removes hosting and allows you to point your DNS outside of DreamHost
