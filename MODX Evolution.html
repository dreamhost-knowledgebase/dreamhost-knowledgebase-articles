As part of an effort to better maintain and support One Click Install applications, DreamHost is no longer able to provide new installations or upgrades for MODX Evolution through the DreamHost panel. The following provides information on how to maintain and where to find further information on upgrading to the latest version.

How do I continue to maintain MODX Evolution?

In order to ensure your MODX Evolution installation remains secure and compatible with the latest versions of PHP, DreamHost strongly recommends bookmarking and regularly checking the official MODX Evolution site, here.

How can I find the latest version of MODX Evolution?

The last version offered by DreamHost was 1.0.15, and the latest version available from the developer is 1.2.1 (which you can download here). It is advisable to upgrade as soon as possible using the instructions from MODX, here.
