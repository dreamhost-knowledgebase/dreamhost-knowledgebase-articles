Overview

If you have several domains and users configured in your panel, you can easily filter them to locate a specific website user. The panel lets you filter your website users by the following:

  • Domains
  • Servers
  • Type
 

Filter users by a domain

This option is useful if you have many domains and need to find the username assigned to one of them quickly. To locate a username assigned to a domain:

  1. Navigate to the SFTP Users & Files page.
  2. In the top menu, click the Filters button.
    A pop-up appears.
  3. Select the Domains filter.
  4. Select a domain name from the list.
    The username assigned to the domain appears.
 

Filter users by their server

This option is useful if you have more than one type of server on your account. For example, a shared server and a VPS. To locate a username assigned to a server:

  1. Navigate to the SFTP Users & Files page.
  2. In the top menu, click the Filters button.
    A pop-up appears.
  3. Select the Servers filter.
  4. Select a server from the list.
    Usernames assigned to this server appear.
 

Filter users by the type of user

This option is useful if you have several usernames created and need to quickly identify which type they are.

Before you assign a domain to a user, you should consider what type of use it may need. This could be either an SFTP or Shell user. The following article explains more about the differences between these two user types:

  • Difference between SFTP and Shell users

To view a list of either SFTP or Shell users:

  1. Navigate to the SFTP Users & Files page.
  2. In the top menu, click the Filters button.
    A pop-up appears.
  3. Select the Type filter.
  4. Select either SFTP or SSH to display a list of this type of user.
