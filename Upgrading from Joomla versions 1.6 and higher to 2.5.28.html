Upgrading from 1.6.x to 2.5.28

  1. Create backups of the both the domain files as well as the database.
  2. Navigate to your Joomla Admin page and go to Extensions -> Extensions Manager, and then click the Update tab.
  3. Click the Find Updates Icon. A list of updates appears.
  4. Place a check mark next to the version, and then click the orange Update button. The system will work for a minute or two.

If you end up with a blank screen, you can either refresh the browser or update by doing the following:

  • Go to the following URL to reload the installer page: administrator/index.php?option=com_installer
  • Once complete, navigate back to the Update tab and update again.
  • The update will run again for a few minutes and then you'll see a message stating "Updating file was successful".

Upgrading from versions 2.5.x and higher to Joomla! 2.5.28 using the DreamHost web panel

You can upgrade from 2.5.x to 2.5.28 (stable version) using the DreamHost panel.

  1. In the panel, navigate to the (Panel > 'Goodies' > 'One-Click Installs') page.
  2. Click Manage Installed Applications.
  3. Click the green icons next to the Upgrades Needed message, and then click the action button for each install of Joomla 2.5 and higher. View the How to upgrade a One-Click Install article for further details.
