Overview

Do not use these instructions unless explicitly instructed by the DreamHost Professional Services Team. The following information is only to be used in a particular phase of a WP migration being done for you by that team; otherwise, the steps will be out of context and can lead to errors if implemented.

In order to initiate a migration service, you must provide DreamHost with access to your WordPress account.

The following describes how to safely create a DreamHost user in your WordPress admin dashboard so that the DreamHost Migrations team can then get started. DreamHost will then be able to log in using an email address, which ensures security by you not needing to share any of your private data.

Creating a new user

To create a new user:

  1. Log in to your WordPress admin account at your current host.
    The WordPress admin dashboard opens. Your dashboard may appear differently from the following image depending on which WordPress version you use, but you should be able to see the following:
  2. In the left-side menu, click Users > Add New and then complete the following:
    1. Enter 'dreamhost' for the username
    2. Enter 'migrate@dreamhost.com' for the email address
    3. Check the checkbox to the right of 'Send User Notification'
    4. Set the role to 'Administrator'
  3. When finished, click the Add New User button.
