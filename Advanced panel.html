Overview

The Advanced menu contains links for other parts of the DreamHost panel that offer additions to your website. These can be used to optimize your overall experience.

MySQL Databases

The MySQL Databases page is where you can add/manage a database which may be used with your website.

View the following article for details:

  • MySQL overview

Subversion

The Subversion page is where you can add/manage a subversion repository. Subversion is a "version control system" to help you manage projects ("repositories") that include many files and/or contributors.

View the following article for details:

  • Subversion overview

Cron Jobs

The Cron Jobs page is where you can add/manage a cron job. Cron is a way of performing scheduled tasks on UNIX systems. Each entry in a user's "crontab" file is called a "cronjob."

View the following article for details:

  • Crontab overview

Jabber IM

On the Jabber IM page, you can add Jabber users to communicate via instant message.

Jabber is an open-source instant messaging platform that uses open, XML-based protocols to create the standard functionality people expect of an IM system: one-to-one chat, multi-user chat, the ability to subscribe to someone else's presence, and so on. The great thing about Jabber on DreamHost is that you can host your OWN Jabber chat server on your domain.

View the following article for details:

  • Jabber overview

Htaccess/WebDAV

The Htaccess/WebDAV page is where you can enable password protection or WebDAV.

You can use this web interface to create usernames and passwords for your friends and associates to access password-protected sections of your websites. You can even use an .htaccess file to help prevent hotlinking of your website files by outside parties or forbid specific file extensions.

The panel option is only able to add the features mentioned above. If you need to add any other code to an .htaccess file, you must create it manually. View the following article for details:

  • .htaccess overview

And view the following for an overview on WebDAV:

  • WebDAV overview

Block Spiders

On the Block Spiders page, you can create an .htaccess file to block specific search engines.

View the following article for details:

  • Bots spiders and crawlers

Anonymous FTP

The Anonymous FTP page is where you can create an FTP service which allows anonymous users the ability to upload to your website.

For further information view the following article:

  • Anonymous FTP

Mongrel and Proxy

These features are currently available for DreamHost VPS and Dedicated servers only. From the Mongrel and Proxy page, you can create a Mongrel application server to host your Ruby web applications, including Ruby on Rails. When you create a Mongrel server, an appropriate proxy server is automatically created and configured for you, and you can also create your own proxy as well.

For more information and setup instructions, please visit the following articles:

  • Mongrel
  • Proxy Server

Remap Sub-Dir

The Remap Sub-Dir page allows you to remap a directory, which is useful for giving other users access in order to alter specific subdirectories without needing to give them full access to the site.

For more information, view the following article:

  • Remap Sub-Directory

Site Statistics

You can enable site statistics on the Site Statistics page. The site statistics application runs through a program called Analog and provides various types of reports, including statistical reports for your domain.

View the following links for further details:

  • Statistics Overview
  • Analog
  • Accessing Stats
