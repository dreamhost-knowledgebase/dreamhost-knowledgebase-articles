DreamHost offers two Cloud services which are available in the panel:

  • DreamObjects – (Panel > ‘Cloud Services’’ > ‘DreamObjects’)
  • DreamCompute – (Panel > ‘Cloud Services’ > ‘DreamCompute’)

DreamObjects

DreamObjects is a cost-effective cloud storage service, which you can use to host static data for your websites, store backups, or develop the next big thing. You can access DreamObjects in your panel using the built-in interface, programmatically via standard APIs, or with a growing library of applications. It’s even compatible with the Amazon S3 API.

View the following article for a list of compatible applications:

  • DreamObjects Compatible Applications

DreamCompute

DreamCompute is a highly scalable cloud computing service built to power everything from:

  • Web and mobile applications
  • Digital media and e-commerce websites
  • Big data analytics
  • Testing and development environments

All instances can be managed through the DreamCompute’s dashboard built off of Openstack’s Horizon. To begin DreamCompute, you can start the service in your panel on the (Panel > ‘Cloud Services’ > ‘DreamCompute’) page.
