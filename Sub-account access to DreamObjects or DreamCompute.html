Overview

Account Privileges allow you to grant account access to another person so they can edit settings within the panel. For example, you could allow someone to access  the following settings.

  • Access to servers settings
  • Access to domain hosting plans
  • Access to domain registrations
  • Access to users

Account privileges allow access to most sections of the panel, however, not all sections are available to all users.

Sub-account access to Cloud Services

It is currently not possible to assign Account privileges to DreamObjects or DreamCompute. This means that when you create a new sub-account, the user will see the following when they log into their account and attempt to access the Cloud Services menu.

It is currently only possible for the primary account owner to access the Cloud Services menu.
