The easiest way to generate a key pair is to have the DreamCompute dashboard generate one for you.

  1. Navigate to the (Panel > 'Cloud Services' > 'DreamCompute') page.
  2. Click theÂ 'View Dashboard' button.
  3. On the left, click 'Compute'.
  4. From the dropdown click 'Key Pairs'.
  5. On the top right click the 'Create Key Pair' button.
  6. Give the key a name, and click 'Create Key Pair'.
  7. Your browser will start downloading the private key. By default it will have the same name you entered in the 'Key Pair Name' field and have a .pem file extension, but you can rename the file to whatever you want.
