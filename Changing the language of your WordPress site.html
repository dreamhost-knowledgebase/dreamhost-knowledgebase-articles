Overview

When you originally install WordPress using the One-Click Installer, the language is set to English. This article explains how to change the default language of your WordPress site so your WordPress dashboard and all of its links and buttons are in the new language.

Changing the language

  1. Log into your WordPress site at example.com/wp-admin.
  2. Navigate to the 'Settings > General' page.
  3. Next to Site Language, click the dropdown menu.
  4. Select the language you want WordPress to use.
  5. Scroll down and click the Save Changes button.

Your panel is now configured to use the new language.
