Overview

The following is a brief overview of Sectigo (formerly Comodo), and how it affects your certificate.

What is Sectigo?

Sectigo is simply the new name of Comodo.

On November 1, 2018, Comodo announced that the company is rebranding to Sectigo. For more information about this, please see the following announcement at Sectigo's website:

  • Comodo CA Is Now Sectigo

How does this affect my certificate?

The new name does not affect your certificate at all.

However, any Comodo certificate issued after January 2019 will reference Sectigo as the certificate authority, while any certificates issued prior to that will still reference Comodo. These are still the same company, and the only difference is the name.
