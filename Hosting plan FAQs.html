General

What type of hosting plans do you offer?

DreamHost offers Shared, VPS, and Dedicated Servers to host your sites. There is also a dedicated WordPress service called DreamPress available. Visit the following article for more information:

  • What types of hosting plans are available?

Is cPanel available at DreamHost?

No. DreamHost's panel was built in-house and is currently the only panel available. Please view the following article for further information:

  • Where can I find cPanel?

How many sites can I set up with a hosting plan?

It depends on the plan. For Shared, VPS, and Dedicated Hosting, you can add as many domains as you like.

For DreamPress, each plan is only set up for a single domain, so if you want multiple domains on DreamPress, you must purchase a DreamPress plan for each domain.

How do I backup my domain?

There are several ways to backup your data. Visit the following article for further details:

  • Backup

Does DreamHost provide long-term backups?

DreamHost keeps on average 2 weeks of website backups and 5 days of database backups. Long-term backups are not included in any hosting plan. However, DreamHost offers DreamObjects, which is a cloud storage service that allows you to back up any data you like at an affordable rate.

Server info

What operating system is used on the web servers?

Shared web servers run Apache.

VPS and Dedicated Servers can run Apache or Nginx.

Do I get root access to my web server?

Root access is not available on a shared, VPS, or DreamPress hosting plan. However, you can create an admin user on a Dedicated server which has sudo/root access. See the following article for more information:

  • Add users to a Dedicated server

Does DreamHost support Windows ASP, .NET, or FrontPage extensions?

No. DreamHost is completely a Linux environment.
