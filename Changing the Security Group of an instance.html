When you originally launch a new instance, you have the option to assign an existing security group to it.

  • How to launch an instance in the dashboard

However, what if you've already created the instance as it's assigned to the default security group? You can then edit the instance and assign it to any existing group you like.

Assigning an existing instance to a security group

  1. Log into your DreamCompute panel.
  2. Navigate to the (Compute > Instances) tab.
  3. To the right of your instance, click the dropdown menu and select 'Edit Security Groups'.
  4. A pop-up box displays. Click the 'Security Groups' tab.
  5. On the left you'll see a list of existing Security Groups. In the above example, the existing Security Group is named 'Cloud'. To the right are groups assigned to this specific instance. In the above example, the current Security Group assigned to the instance is 'default'.
  6. Click the + button next to the group on the left you wish to add to your instance.
  7. When your chosen Security Group appears on the right side under 'Instance Security Groups', click the Save button.

Your instance now has a new Security Group assigned to it.
