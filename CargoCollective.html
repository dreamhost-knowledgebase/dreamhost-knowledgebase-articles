Overview

CargoCollective is a personal publishing platform that helps artists create simple websites to display their work. View more at the following link:

  • http://cargocollective.com/

Pointing your DreamHost site to CargoCollective

If your domain is registered at DreamHost, you can point your DNS to CargoCollective. View the following link for instructions:

  • Pointing to CargoCollective

There are two options: point the primary domain, or point a subdomain such as blog.example.com.

Pointing a primary domain

The following instructions show you how to point your site example.com to CargoCollective.

The screenshots below use the domain example.com. Your domain will appear instead when following these instructions.

  1. Set your domain up in your panel to DNS Only.
    Name (leave blank)
    Type A
    Value 173.203.204.123
  2. In your DreamHost panel, create an A record for your domain.
  3. Save this record by clicking the Add Record Now! button.
    Name www
    Type A
    Value 173.203.204.123
  4. Repeat this step for the 'www' A record.

Your site is now pointed to CargoCollective. Due to DNS propagation, it can take several hours for the records to update online.

Pointing a subdomain

The following instructions show you how to point your site blog.example.com to CargoCollective using a subdomain.

  1. Set your domain up in your panel to DNS Only.
    Name (leave blank)
    Type A
    Value 173.203.204.123
  2. In your DreamHost panel, create an A record for your domain.
  3. Save this record by clicking the Add Record Now! button.
    Name blog
    Type CNAME
    Value cargocollective.com
  4. In your DreamHost panel, create a CNAME record

Your subdomain is now pointed to CargoCollective. Due to DNS propagation, it can take several hours for the records to update online.
