When you no longer need an instance, you can delete it using the OpenStack command line client.

View all instances

List all instances using the list command.

[user@localhost]$ openstack server list
+-------------+----------------------+--------+------------+-------------+------------------+
| ID          | Name                 | Status | Task State | Power State | Networks         |
+-------------+----------------------+--------+------------+-------------+------------------+
| 8a99547e... | myInstanceFromVolume | ACTIVE | None       | Running     | private=10.0.0.4 |
| d7efd3e4... | newServer            | ERROR  | None       | NOSTATE     |                  |
+-------------+----------------------+--------+------------+-------------+------------------+

Delete the instance

Run the delete command to delete the instance. The following example deletes the newServer instance which is in ERROR state:

[user@localhost]$ openstack server delete newServer

The command does not notify that your server was deleted.

Verify it was deleted

To verify that the server was deleted, run the list command as shown above. The instance should no longer show in the list
