Piwigo is open source photo gallery software for the web. It contains many features that assists with exhibiting and organizing your photography.

Photos can be added via the administrative panel or any FTP client. Photo upload capabilities have been extended to be compatible with desktop applications like digiKam, Shotwell, Lightroom, iPhoto, Aperture, and also iPhone/Android applications.

Best of all, Piwigo is open source and free to use.

Piwigo 2.7.4, released in Feb 2015, is available as a One-Click Install from the (Panel > 'Domains' > 'One-Click Installs') page of your panel.

The One-Click version available in the DreamHost panel is only compatible with PHP 5.6. If you wish to use this version, you must change the PHP version of your site to 5.6. View the following article for instructions on how to do this:
