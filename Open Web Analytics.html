Overview

Open Web Analytics (OWA) is open source web analytics software you can use to track and analyze how people use your web sites and applications. OWA is licensed under GPL and provides website owners and developers with easy ways to add analytics to their sites using simple JavaScript, PHP, or REST based APIs. If you would like more information in regards to Open Web Analytics, please visit their website here:

  • http://www.openwebanalytics.com/
