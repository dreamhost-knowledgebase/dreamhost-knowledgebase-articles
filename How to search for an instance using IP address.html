You can use the OpenStack command line client to search for an instance using the IP address parameter. For example, the following example shows the results of a search of the IP address 10.0.0.4.

[user@localhost]$ openstack server list --ip 10.0.0.4
+------------------+----------------------+--------+------------+-------------+------------------+
| ID               | Name                 | Status | Task State | Power State | Networks         |
+------------------+----------------------+--------+------------+-------------+------------------+
| 8a99547e-7385... | myInstanceFromVolume | ACTIVE | None       | Running     | private=10.0.0.4 |
+------------------+----------------------+--------+------------+-------------+------------------+
